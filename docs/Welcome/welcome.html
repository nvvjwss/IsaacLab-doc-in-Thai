Overview

ยินดีต้อนรับสู่ Isaac Lab!

Isaac Lab คือ framework สำหรับ robot learning ที่รวมทุกอย่างไว้ในที่เดียวและมีความ modular สูง โดยมีเป้าหมายเพื่อทำให้ workflows ทั่วไปในการวิจัยด้าน robotics ง่ายขึ้น เช่น reinforcement learning, learning from demonstrations และ motion planning โดยสร้างขึ้นบน NVIDIA Isaac Sim เพื่อใช้ประโยชน์จากความสามารถ simulation ล่าสุดสำหรับ photo-realistic scenes และการ simulate ที่รวดเร็วและมีประสิทธิภาพ

เป้าหมายหลักของ framework มีดังนี้:

  • Modularity: ปรับแต่งและเพิ่ม environments, robots และ sensors ใหม่ได้ง่าย
  • Agility: ปรับตัวให้เข้ากับความต้องการที่เปลี่ยนแปลงของ community
  • Openness: เป็น open-source เพื่อให้ community สามารถ contribute และขยายต่อได้
  • Batteries-included: มี environments, sensors และ tasks จำนวนมากที่พร้อมใช้งานทันที

features หลักที่มีใน Isaac Lab ได้แก่ physics simulation ที่รวดเร็วและแม่นยำโดย PhysX, tiled rendering APIs สำหรับ vectorized rendering, domain randomization เพื่อเพิ่ม robustness และความสามารถในการปรับตัว และรองรับการรันบน cloud

นอกจากนี้ Isaac Lab ยังมี environments หลากหลายประเภท และกำลังเพิ่ม environments ใหม่อย่างต่อเนื่อง ได้แก่ classic control tasks, fixed-arm และ dexterous manipulation tasks, legged locomotion tasks และ navigation tasks

Robots ที่พร้อมใช้งาน

Isaac Lab มี robot assets ที่ฝังมาพร้อมกับ platform และพร้อม train ได้ทันทีดังนี้:

  • Classic: Cartpole, Humanoid, Ant
  • Fixed-Arm and Hands: UR10, Franka, Allegro, Shadow Hand
  • Quadrupeds: Anybotics Anymal-B, Anymal-C, Anymal-D, Unitree A1, Unitree Go1, Unitree Go2, Boston Dynamics Spot
  • Humanoids: Unitree H1, Unitree G1
  • Quadcopter: Crazyflie
